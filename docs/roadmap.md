# Roadmap

DBForensics is in early MVP / public preview. The roadmap may change based on feedback from DBAs, security engineers, incident responders, digital forensics professionals, and packet analysis users.

## Current Version: v0.1 Preview

Completed:

- MySQL PCAP analysis
- SQL session reconstruction
- SQL event timeline
- Incident summary
- Basic SQL detail view
- Basic protocol detail view
- Offline PCAP / PCAPNG workflow
- Sample PCAP
- Windows desktop MVP

## Next: v0.2

Planned:

- Risk engine
- Slow SQL detection
- CSV export
- Improved search filters
- Better application detection
- Report draft
- Better release packaging
- More sample captures

## Later

Planned:

- PostgreSQL support
- Oracle support
- SQL Server support
- Report generation
- Markdown / HTML report export
- Real-time capture
- CLI version
- Rule configuration
- Enterprise licensing

## Not Planned Yet

Not planned for the current product direction:

- Cloud SaaS
- Agent-based monitoring
- Database proxy
- Full DAM replacement
- Query firewall
- Production traffic blocking
- Hosted PCAP storage

## Feedback Wanted

The most useful roadmap feedback includes:

- Which database protocols you need most
- Whether you analyze offline PCAPs or need live capture
- What fields you expect in investigation reports
- What search filters matter in real incidents
- How you currently review database traffic during incidents
